# Random Task Generator
**Автор:** Котков Максим
Приложение для борьбы с прокрастинацией. Выбирает случайную полезную задачу из списка и сохраняет историю ваших действий.
- Генерация случайных задач по категориям.
- Фильтрация истории (Спорт, Учеба, Работа).
- Автоматическое сохранение истории в JSON.

## Использование
1. Запустите `main.py`.
2. Нажмите кнопку "Сгенерировать задачу".
3. Используйте выпадающий список для фильтрации выполненных дел.
